Research

I have done several works on mutli-agent reinforcment learning, curriculum learning and multi robot collaboration. Representative papers are highlighted.

Optimistic Multi-Agent Policy Gradient for Cooperative Tasks
Wenshuai Zhao, Yi Zhao, Zhiyuan Li, Juho Kannala, Joni Pajarinen
arXiv, 2023
code / arXiv

In order to overcome the relative overgeneralization problem in multi agent learning, we propose to enable optimism in multi-agent policy gradient methods by reshaping advantages. This method is easy to implement and outperforms strong baselines on popular MARL benchmarks.

Less Is More: Robust Robot Learning via Partially Observable Multi-Agent Reinforcement Learning
Wenshuai Zhao*, Eetu-Aleksi Rantala*, Joni Pajarinen, Jorge Pena Queralta
arXiv, 2023
code / arXiv

Exploring partial observability in Multi-Agent Reinforcement Learning. Our experiments show that in many multi agent systems where agents are weakly coupled, partial observation can still enable optimal decision making. Moreover, in a mobile robot manipulator, we show partial observation of agents can improve robustness to agent failure.

Self-Paced Multi-Agent Reinforcement Learning
Wenshuai Zhao, Zhiyuan Li, Joni Pajarinen
arXiv, 2023
code / arXiv

We show two flaws in existing reward based curriculum learning algorithms when generating number of agents as curriculum in MARL. Instead, we propose a learning progress metric as a new optimization objective which generates curriculum maximizing the learning progress of agents. Our method SPMARL outperforms baselines by a faster convergence and generating interpretable curriculum.
